# Statistiques Appliquées et Analyse de Données
## De la Théorie à la Pratique

## Description
Projet réalisé dans le cadre du référentiel **Data Analyst - INT Maroc**.

L'objectif est de maîtriser les concepts statistiques fondamentaux 
et de les appliquer sur des datasets réels afin de produire des 
analyses rigoureuses, interprétables et orientées métier.

## Datasets utilisés
- `healthexp.csv` — Données de santé publique mondiale
- `mpg.csv` — Données automobiles

## Structure du projet
- `data/` : datasets utilisés
- `notebooks/` : analyses Jupyter par phase
- `images/` : visualisations exportées

## Phases du projet

### 🧠 Phase 1 — Compréhension & Documentation
Documentation des concepts statistiques :
- Statistiques descriptives
- Population vs Échantillon
- Probabilités et distributions
- Théorème Central Limite
- Tests d'hypothèses

### ⚙️ Phase 2 — Analyse Dataset Santé
- Analyse des variables Spending_USD et Life_Expectancy
- Évolution temporelle des dépenses de santé
- Corrélation entre dépenses et espérance de vie
- Test d'hypothèse sur la durée de vie aux États-Unis

### 🚗 Phase 3 — Analyse Dataset MPG
- Analyse des variables mpg, cylinders, origin
- Relation horsepower vs consommation
- Évolution du poids des voitures américaines (70-82)
- Tendances générales par origine et par année

## Technologies utilisées
- Python (pandas, matplotlib, seaborn)
- Jupyter Notebook
